Le Socle Interministeriel des Logiciels Libres

Retrouvez les logiciels libres recommandés au Burkina Faso ...

Le SILL est une liste de logiciels libres significativement utilisés dans des structures de la fonction publique et qui forment un catalogue de référence pour les usagers de l’administration

La liste des logiciels du SILL est publiée dans un site web de l’administration publique. Le Site web permet l’identification rapide d’un logiciel, la dernière version stable recommandée et un historique des anciennes versions stables (toute administration est libre d’utiliser une version plus récente). Il est donc possible de consulter l’état d’avancement du statut d’un logiciel du SILL, selon qu’il est en cours d’observation ou recommandé. Un lien de téléchargement est disponible sur le site.
